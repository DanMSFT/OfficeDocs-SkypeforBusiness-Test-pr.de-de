# Cmdlets für Persistent Chat Server

 

_**Letztes Änderungsdatum des Themas:** 2012-06-27_

Mit dem Cmdlet für beständige Chatsitzungen können Sie den Microsoft Lync Server 2013 Persistent Chat-Dienst (früher als Gruppenchat-Dienst bezeichnet) konfigurieren und verwalten. Der Persistent Chat ermöglicht Benutzern die Teilnahme an Online-Sofortnachrichtensitzungen. Diese Sitzungen können in Echtzeit stattfinden, ihr Inhalt wird jedoch dauerhaft gespeichert. So können sie jederzeit von jedem Benutzer wieder aufgenommen werden.
