# Microsoft Lync Server 2013

 

_**Letztes Änderungsdatum des Themas:** 2016-12-08_

Die Microsoft Lync Server 2013Kommunikationssoftware bietet Lösungen für Instant Messaging (Sofortnachrichten), Anwesenheitsinformationen, Konferenzen und Telefonie, die die Anforderungen an die Zusammenarbeit auf Unternehmensebene erfüllen.

Möchten Sie eine Kopie der Lync Server 2013-Dokumentation herunterladen? Sie können die kompilierte **LyncServer2013\_ITPro**-Hilfedateil (.chm) aus dem Microsoft Download Center unter [http://go.microsoft.com/fwlink/p/?LinkID=331103](http://go.microsoft.com/fwlink/p/?linkid=331103) herunterladen.

In der folgenden Tabelle sind die Themen in der Lync Server 2013-Dokumentationsbibliothek aufgelistet.
